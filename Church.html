GL Pos-Church

A solution for selecting the Church offering as well as collection of donation / annadhanam from pilgrims for places of worship from any location
grid_view
Components
Church Software application is an Android based app which provides the much needed offerrings at POS terminals.
The solution helps Church, trusts, etc to record and generate bill at the integrated to the POS terminal payment gateway including donations and annadhanam. Single religious organization can have multiple terminals working from different remote locations.
brush
Design
The application is designed to run on Android on cloud based server with dedicated DB running at server as well as on the device which
replicates frequently. For customized solutions, even single database can serve multiple POS terminals concurrently and consolidation at desktop to monitor revenues from different terminals and payment modes are possible.
code
Less Code
The application has been coded, designed, fine tuned, rested and optimized with respect to hardware resources for effective performance
devices
Fully Responsive
All application frontends are developed in responsive mode to cater comfort for end customers.
